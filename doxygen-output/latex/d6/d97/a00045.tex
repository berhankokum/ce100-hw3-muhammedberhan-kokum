Create a dictionary from the list of assembly steps for easy lookup by step ID.

Create a hash set to keep track of the steps that have been visited during the depth-\/first search (DFS).

Create a stack to hold the sorted steps. Steps are added to the stack after all their dependencies have been visited.

Iterate over each step in the dictionary. If a step has not been visited, perform a DFS from that step.

Return the steps in the order they should be performed. Since we added steps to the stack after their dependencies,

converting the stack to a list gives the correct order of steps.
