HuffmanNode ce100_hw3_algo_lib_cs.HuffmanCoding.BuildHuffmanTree ( Dictionary< char, int >  frequencies)
inlineprivate

Create a priority queue of Huffman nodes, each node is associated with a character and its frequency

Continue until there is only one node left in the priority queue (the root of the Huffman tree)

Order the priority queue by frequency

The two nodes with the lowest frequency are combined into a new node

Create a new node with the combined frequency of the two lowest nodes, and these two nodes as its children

Remove the two lowest nodes from the priority queue

Add the new node to the priority queue

Return the remaining node in the priority queue, which is the root of the Huffman tree
